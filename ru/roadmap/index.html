Дорожная карта развития BTG

В Bitcoin Gold мы стремимся построить открытый и
ориентированный на сотрудничество мир. Мы считаем, что
мощная криптовалюта должна быть не только быстрой и
полнофункциональной, но также должна быть фундаментально
безопасной и децентрализованной. Эти убеждения формируют
уникальную дорожную карту Bitcoin Gold.
На этой странице отражен прогресс в разработке проекта Bitcoin
Gold и планы на будущее. Данные будут регулярно обновляться,
так как мы предоставляем сообществу отчеты о ходе работы
каждые две недели.

Complete

Open Source
Library: bitcore

  • 100% 100%
Bitcore: инфраструктура для развития Bitcoin Gold и приложений
на блокчейне. Добавлена поддержка BTG для bitcore-node,
bitcore-lib и bitcore-deb. Шаг, необходимый для интеграции.

Open Source
Library: insight

  • 100% 100%
Веб-сервис блокчейн обозревателя с API интеграцией на Bitcore
Node. Наш обозреватель предоставляет общедоступный API
для взаимодействия с основной цепью, бесплатный для частных
лиц и профессиональных сервисов (таких как Exodus).

Open Source Library: bitcoinjs-lib

  • 100% 100%
Связанные с Bitcoin функции реализованы на JavaScript – это
позволяет разработчикам приложений всех видов (включая
обменники) и веб-сервисов легко использовать BTG.

Open Source
Library: BitcoinJ

  • 100% 100%
Java-реализация протокола Bitcoin от bitcoinj – важный
“строительный” блок для разработчиков Java проектов, таких как
кошелек MyCelium.

Electrum Wallet
& Bech32

  • 100% 100%
ElectrumG – наша версия кошелька Electrum с широкими
возможностями, полной поддержкой Bech32 (SegWit), HWкошельков и т.д. Этот легкий SPV-кошелек функционирует на
нашиих бесплатных серверах ElectrumX.

Lightning Network Technical Preview

  • 100% 100%
Полная техническая апробация LN на тестовой сети, включая
интерфейс lnd (LIghtning Network Daemon от Lightning Labs) для
BTG, recksplorer (LN обозреватель) и lseed (LN загрузчик DNS).

Smart Contract
Research

  • 100% 100%
Заключение исследования: наилучший подход заключается в
развертывании Тьюринг-полного языка контрактов на сайдчейне
Plasma поверх главной цепочки BTG. Plasma для BTG станет
топовым проектом 2019 года.

Active

LN: Mainnet

  • 0% 0%
Развертывание стабильной BTG версии lnd в основной сети.
Могут быть запущены и другие реализации LN, включая clightning (Blockstream) и eclair (ACINQ)

Интерфейс btcd / btcsuite

  • 80% 80%
btcd: альтернативная Полная Нода на Go, основа для полного
набора инструментов btcsuite. Активно используется lnd и
Neutrino (от Lightning Labs) и является условием для разработки мобильных LN кошельков.

LN: кошелек для ПК

  • 0% 0%
Поддержка BTG LN для превосходного кроссплатформенного
кошелька Zap, который обеспечивает дружественный
пользовательский интерфейс. (Поддержка LN планируется в
ближайшем будущем и для Electrum.)

LN: кошелек для мобильных устройств

  • 0% 0%
Перенос LN на мобильные устройства будет иметь ключевое
значение для широкого внедрения, поэтому мы интегрируем BTG в RawTX (для чего требуется Neutrino).

BTCPay Интеграция BTG Lightning Network

  • 30% 30%
BTCPay (лучший децентрализованный платежный сервис) уже
поддерживает BTG, поэтому расширение BTCPay для
поддержки BTG LN будет следующим шагом.

Подписи Шнорра

  • 0% 0%
Многие транзакции включают отправку нескольких входных
потоков данных на один платежный адрес, и каждый входной
поток требует подписи. Подписи Шнорра позволяют
подписывать несколько входных потоков одной подписью,
существенно экономя место в блокчейне.

Техническое описание BTG Plasma

  • 0% 0%
Этот амбициозный проект начнется с технического описания
протокола, основанного на расширении существующих
реализаций Plasma.

Plasma: реализация Mainchain

  • 0% 0%
Изменения основной сети, включая новые опкоды, должны быть
протестированы и развернуты для поддержки Plasma.

Плазма: реализация Sidechain

  • 0% 0%
Фактическое развертывание сайдчейнов, поддерживаемых
основной цепью. Это обеспечит огромную пропускную
способность и быстрое выполнение кода на сайдчейне Plasma,
одновременно обеспечивая безопасность средств на основном
блокчейне.

Кошелек MyCelium

  • 0% 0%
Известный кошелек MyCelium для Android и iOS добавляет ряд
интересных функций, включая Local Trader для обмена монет.
Добавление поддержки BTG откроет доступ к этим
возможностям для BTG сообщества (основано на библиотеке
BitcoinJ).

CCBN – Cross Chain Block Notarization

  • 0% 0%

A decentralized approach to thwart secret mining attacks by notarizing blocks to independent blockchains, providing a timing reference used to resolve chain splits in favor of honest mining.

Future

Исследовательская поддержка активов, созданных

пользователями на базе основного блокчейна Активы, создаваемые пользователями, обеспечивают
поддержку бизнес-моделей по предоставлению различных типов
сервисов, работающих на основе токенов.

Поддержка атомарных свопов

BTG будет поддерживать технологию обмена криптовалют, но
ее использование будет зависеть от прочих сервисов,
предоставляющих услуги по сопоставлению заявок на обмен.

Конфиденциальные транзакции

Конфиденциальные транзакции защищают коммерческую тайну,
а также обеспечивают личную неприкосновенность. Когда и как
обеспечивать конфиденциальность, остается открытым
вопросом.

Децентрализованные майнинговые пулы

Текущие майнинговые пулы являются централизованными, к
ним требуется доверие; децентрализованные пулы предлагают
более открытую и децентрализованную альтернативу.

Блокчейн-демократия

Долгосрочная работоспособность проекта BTG зависит как от
хорошего управления, так и от способности сообщества
выражать свои намерения без спорных хардфорков.
